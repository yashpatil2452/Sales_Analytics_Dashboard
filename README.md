# 📊 Sales Analytics Dashboard (Excel)

## 📌 Project Overview
This project is an interactive Sales Analytics Dashboard built entirely in Microsoft Excel. It helps analyze sales performance, customer behavior, and business trends using dynamic visuals and slicers.

## 🎯 Objectives
- Monitor overall sales performance
- Analyze trends over time
- Understand customer segments (new vs repeat)
- Identify top-performing services, cities, and customers

## 🛠️ Tools & Technologies Used
- Microsoft Excel – Dashboard creation
- Pivot Tables – Data aggregation and analysis
- Pivot Charts – Data visualization
- Slicers – Interactive filtering
- Data Cleaning – Preprocessing and structuring data

## 📂 Dataset Description
The dataset includes:
- Sales data (revenue, margin, etc.)
- Service categories (Ads, SEO, App Dev, Social Media, Web Dev)
- Customer data (new and repeat customers)
- Location data (city-wise sales)
- Time-based data (monthly/yearly trends)

## 📊 Dashboard Features

### 1. Interactive KPI Cards
- Displays key metrics like total sales, margin, and customers
- Automatically updates based on slicer selections

### 2. Sales Trend Analysis
- Line chart showing sales trends over time
- Helps identify growth patterns and seasonality

### 3. Customer Source Analysis
- Breakdown of customer acquisition channels (Ads, Organic, Walk-in)
- Comparison across different time periods

### 4. Sales by City
- Shows contribution of different cities to overall sales
- Helps identify high-performing locations

### 5. Sales by Service Type
- Category-wise comparison of services
- Useful for understanding business focus areas

### 6. Department Margin Analysis
- Displays contribution of different departments to overall margin

### 7. New vs Repeat Sales
- Compares new and returning customers
- Helps analyze customer retention

### 8. Top Customers
- Highlights top revenue-generating customers

## 🔍 Key Insights
- Sales performance varies across categories, locations, and time
- Customer retention significantly impacts overall revenue
- Certain services and cities consistently perform better
- Trends and patterns can be identified using time-based analysis

## 🚀 How to Use
1. Open the Excel file (.xlsx)
2. Use slicers (e.g., Year filter)
3. Interact with charts and pivot tables
4. Analyze dynamically updated insights
